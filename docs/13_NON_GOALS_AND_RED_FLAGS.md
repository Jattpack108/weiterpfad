# 13 — NON-GOALS AND RED FLAGS

## Non-goals
- not a complete live directory of every provider
- not a public-sector information clone
- not a compliance-heavy backend product in phase 1
- not an app-store-style marketplace
- not an overbuilt dashboard product

## Red flags
- too much copy on every page
- bright generic marketing visuals
- too many sections with weak purpose
- fake complexity
- fake metrics
- fake testimonials
- fake live capacity
- vague AI wording
- inconsistent category naming

## If Codex has to choose
Choose:
- fewer, stronger sections
- cleaner code
- better hierarchy
- more whitespace
- more believable placeholder content

Do not choose:
- more features
- more animations
- more decorative visuals
- more filler text
