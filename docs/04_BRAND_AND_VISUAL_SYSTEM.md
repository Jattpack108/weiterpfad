# 04 — BRAND AND VISUAL SYSTEM

## Visual direction
The product should feel:
- premium
- calm
- spacious
- serious
- soft-edged but not soft-minded
- modern without trend-chasing
- digital without coldness

## Visual anti-patterns to avoid
- bright generic SaaS gradients
- over-animated cards
- too many colors
- too many badges
- startup clichés
- flat corporate government aesthetics
- “AI magic” visuals
- overly cute disability branding

## Good visual cues
- dark or muted base
- subtle depth
- restrained highlights
- strong spacing
- elegant rounded geometry
- typography with authority and calm
- nature-adjacent accents without becoming literal

## Why the homepage reference works
The existing page already establishes:
- a strong dark calm base
- restrained glow and depth
- a premium hero
- a visible matching demo
- provider and roadmap credibility

Codex should preserve this overall spirit while improving architecture and multi-page coherence.

## Layout principles
- generous whitespace
- clean section rhythm
- no crammed multi-column text blocks unless needed
- cards should be readable, not decorative clutter
- every visual section should earn its existence

## Motion principles
- use motion sparingly
- motion should support clarity, not show off
- subtle entrance, hover and filter state motion is enough

## Typography principles
- strong headings
- readable body copy
- disciplined line length
- no tiny gray text
- avoid fancy display fonts unless already present and tasteful

## Color principles
- base: dark calm neutrals
- accent: natural, muted green / soft light tones
- utility status colors only where necessary
- contrast must remain strong

## Emotional principle
The visual system should lower stress.
