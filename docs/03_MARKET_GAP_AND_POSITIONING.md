# 03 — MARKET GAP AND POSITIONING

## Market reality
There are already existing portals, directories, advisory offers and partial search tools in the space.
The problem is not that nothing exists.
The problem is that no calm, dominant, end-user product seems to integrate the family transition journey well.

## Strategic interpretation
This means Weiterpfad should not try to sound novel by claiming:
- there is no information today
- there are no existing offers
- there is no search at all

Instead Weiterpfad should claim:
- the current search experience is fragmented
- the user journey is not solved
- the decision layer is weak
- comparison and progression are underbuilt

## What not to build
- a copy of an existing portal
- a broad “everything in disability support” site
- a static address directory with filters only
- a claims-heavy AI site

## What to build instead
- a transition-specific discovery and matching layer
- a premium trust-first interface
- a product that starts simple and grows into operational usefulness

## Competitive differentiation
Weiterpfad should differentiate on:
- sharper scope
- better design
- clearer user flow
- better matching logic
- stronger comparison potential
- provider-ready structure
- future availability layer

## Good positioning line
A calmer, clearer path after school.

## Positioning guardrail
The product should feel narrower and more useful, not broader and more abstract.
