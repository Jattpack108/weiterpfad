# README FOR YOU

## What this bundle is
This is the strongest Codex handoff folder for this project.
It is designed so Codex can open one folder and infer:
- what Weiterpfad is
- what problem it solves
- what the market gap is
- what the website should feel like
- what routes must exist
- what features matter now versus later
- how the provider and data strategy should evolve
- what code and design quality bar to hit

## What is already included
- the current visual homepage reference in `app/page.tsx`
- a single master brief
- detailed product, market, UX, content and technical docs
- structured mock data seeds
- brand tokens and design direction

## Best use
Give Codex only the prompt from `PROMPT_FOR_CODEX.txt`.
Let Codex read the folder and build from there.

## Important note
This bundle is a reconstruction from the content that was actually established in the chat.
It is intentionally more explicit than the chat itself so Codex has less room to misread the intent.
